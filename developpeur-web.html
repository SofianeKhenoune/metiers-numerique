Développeur Web

Le développeur web crée des sites internet et des applications : il écrit du code, imagine le design, rend les pages interactives et s’assure que tout fonctionne sur ordinateur, tablette et mobile.

Exemple concret : J’ai créé un site pour une association sportive où les membres peuvent s’inscrire en ligne et voir les résultats des matchs.
Conseil pour les collégiens : Tu aimes les jeux vidéo ? Essaie de créer ton propre mini-jeu en ligne ! Il existe plein de ressources gratuites pour débuter (OpenClassrooms, Grafikart, MDN Web Docs…)
Retour aux métiers

Missions principales

  • Créer des sites vitrines, blogs, boutiques
  • Développer des applications interactives
  • Assurer la maintenance et la sécurité

Qualités utiles

  • Curiosité, logique, créativité
  • Patience et persévérance
  • Goût du travail en équipe

Outils & environnements

  • HTML, CSS, JavaScript, PHP…
  • Frameworks (React, Symfony…)
  • Logiciels de design (Figma…)

Journée type

  • Analyser les besoins d’un projet
  • Écrire et tester du code
  • Corriger des bugs, améliorer le site
  • Collaborer avec d’autres développeurs

Ressources fiables

  • MDN Web Docs
  • W3C - standards du Web
  • OpenClassrooms

Débouchés

  • Entreprises, agences web, freelance
  • Développeur front-end, back-end, fullstack

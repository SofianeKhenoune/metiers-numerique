## Fonctionnalites
- Page d accueil immersive en plein ecran
- Pages metiers dediees (Developpeur Web, Assistant Numerique, Formateur Digital)
- Quiz interactif avec correction et couleurs (vrai/faux)
- Design moderne responsive, mobile first

## Demarrage rapide
1. Ouvrir `index.html` dans un navigateur.
2. Naviguer vers les pages metiers et le quiz.

## Structure du projet
- `index.html` : page d accueil
- `developpeur-web.html` : page metier
- `assistant-numerique.html` : page metier
- `formateur-digital.html` : page metier
- `quiz.html` : page quiz
- `css/style.css` : styles additionnels
- `js/quiz.js` : logique du quiz
- `img/` : illustrations et images

## Quiz
Le quiz se trouve dans `quiz.html` et est corrige par `js/quiz.js`.

## FAQ
**Comment changer les questions du quiz ?**
Modifie les questions dans `quiz.html` et les bonnes reponses dans `js/quiz.js`.

**Comment mettre une capture d ecran ?**
Ajoute une image `img/screenshot.png` et elle s affichera dans le README.

**Comment publier le site ?**
Tu peux utiliser GitHub Pages en pointant sur la branche `main`.
